# KeuanganApp

Aplikasi pencatatan keuangan pribadi berbasis web dengan Laravel. Kelola pemasukan, pengeluaran, mutasi antar akun, dan pantau saldo dalam periode tertentu.

## Fitur

- Manajemen akun/dompet
- Catat pemasukan & pengeluaran per kategori
- Mutasi saldo antar akun
- Periode keuangan (tahunan/bulanan)
- Grafik dan ringkasan dashboard
- Manajemen pengguna (admin)

## Teknologi

- Laravel
- Tailwind CSS
- Alpine.js
- MySQL
